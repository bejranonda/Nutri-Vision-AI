## 1. UX RESEARCH MODELS APPLICATION

### 1.1 Five Core UX Research Models

**Model 1: Persona Creation**
- **Purpose:** Develop detailed user profiles based on research
- **Application:** Define 4 primary personas for NutriVision
- **Outcome:** Design decisions aligned with real user goals, behaviors, pain points

**Model 2: User Journey Mapping**
- **Purpose:** Visualize entire user experience over time
- **Application:** Map touchpoints, emotions, friction points across user lifecycle
- **Outcome:** Identify UX improvements at each stage

**Model 3: Competitive Analysis**
- **Purpose:** Systematically evaluate rivals' UX/UI
- **Application:** Compare 6 competitors across UX dimensions
- **Outcome:** Identify market gaps and best practices

**Model 4: Card Sorting / Tree Testing**
- **Purpose:** Validate information architecture
- **Application:** Test navigation structure with 15-20 Thai users
- **Outcome:** Optimal IA that matches user mental models

**Model 5: Heuristic Evaluation**
- **Purpose:** Expert review against usability principles
- **Application:** Audit NutriVision against Nielsen's 10 heuristics
- **Outcome:** Systematic identification of usability problems
